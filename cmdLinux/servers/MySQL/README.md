  1. Instalar el paquete mysql-server:
~~~bash
sudo apt install mysql-server -y
~~~
  2. Inicio el srvicio.
~~~bash
sudo systemctl start mysql.service
~~~

### La instalación se probó con éxito y funcionó en Ubuntu Server 22.
